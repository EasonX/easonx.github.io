Octopress on Windows Notes

I sold my old iMac last Saturday and now I must rely on Windows box in my office to update my octopress. To prepare the installation, a few pacakges should be installed.

Install applications before hand

  • msysGit Official Git windows build, with a powerful bash terminal.
  • Python 2.7 Pigment support
  • Ruby >= 1.9.2 I use Ruby Installer 1.9.3.
  • Ruby DevKit Some Ruby package such as posix-spawn require build tools.
  • MarkdownPad Windows alternative of Mou
  • Vim Swiss army knife.
  • MinGW First install mingw-get-inst then other parts.

Chinese Characters

According to this Post, I did some modification to show chinese correctly.

  1. Append to C:\Program Files (x86)\Git\etc\git-completion.bash: 

     $ alias ls='ls --show-control-chars --color=auto'

  2. Change options in C:\Program Files (x86)\Git\etc\inputrc: 

     set output-meta on
 set convert-meta off

  3. Edit C:\Program Files (x86)\Git\etc\profile: 

     export LESSCHARSET=utf-8

Follow my Post for normal octopress setting. But if you encounter error like this:

Invalid byte sequence in GB2312 (ArgumentError)

Then please edit ~/.bashrc and append

export LANG="en_US.UTF-8"

SSH keys and git recognition.

run :

$ rake setup_github_pages

The git address should be like:

github.com:username/username.github.com.git

Then execute:

$ git config --global user.name "Your Name Here"
# Sets the default name for git to use when you commit
$ git config --global user.email "your_email@youremail.com"
# Sets the default email for git to use when you commit

After this, we could generate SSH public key.

$ ssh-keygen -C "username@email.com" -t rsa

During this step, you will be asked to offer a passphrase.
Then go to C:\Users\YourUserName\.ssh copy public key file contents and post it to github.com on your account setting, SSH key menu item. You need to offer your GitHub account password for confirmation.

The first time you deploy website:

$ rake deploy

You will be asked for passphrase we offered before.

Happy blogging.
