Crack wifi network

This is just a summary for references, it’s not intended to perform or teach anybody for any illegal action.

We always heard that Wifi network may be cracked, and your neighbor might still your bandwidth for their own. Theoretically it’s possible, but how is it practically?

WEP

Tools needed:

  • aircrack-ng: an 802.11 WEP and WPA-PSK keys cracking program that can recover keys once enough data packets have been captured.
  • airport private framework.

If you want a more user friendly GUI interface, Wifi Crack may help, acting as a wrapper for the 2 tools above. Basically, you just need to use the private framework to sniff packets and try the password. For example, your wifi interface is “en1” and the channel you want sniff is 11, the command shall be:

/System/Library/PrivateFrameworks/Apple80211.framework/Versions/Current/Resources/airport en1 sniff 11

To expose airport to PATH, you can link it to /usr/sbin.

Alternatively, you can use kismac to dump the wifi data

Once you collected enough data, you could use aircrack-ng to decode the WEP wifi. command:

aircrack-ng -a 1 -b ec:1a:59:2b:cd:30 /private/tmp/airportSniff*.cap

the -b is followed by the target router you want to crack. Just open it overnight, and hope you can get the password.

WPA

  • aircrack-ng
  • reaver

It’s roughly the same, other than that at last you shall use reaver to monitor and decode the target router

reaver -i en1 -b ec:1a:59:2b:cd:30 -vv

But please notice that, the airodump command is only available on Linux. It’s not working on OS X. you need to use airport to get the info you need.

sudo airport -s

Tips: Good to know that there is GUI for dumping wifi frames
