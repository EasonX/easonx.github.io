Python script in Excel

I am help our junior labmate to make a easy calculator requiring curve fitting. The interface shall be easy to access and human-friendly. And of course, I don’t want a huge project, and a win32 interface is just some fancy and useless to a specific job. So I decide to write a script using my favorite script language: Python.

The graphic interface I was first considering web, using a server side scripting. But I veto this option quite fast: once after I left the lab, no one can maintain it and maybe no one know how to start the server. To the contrary, Excel will be an excellent interface: easy to handle, and WYSIWYG.

To equip Excel with python, at least under Windows.I did some googling. Especially from these 3 links in stackoverflow (1,2,3). It turns out several options:

  • Pyxll
  • ExcelPython v2
  • pyinex
  • DataNitro
  • IronPython

If I just want to manipulate excel files outside of Excel, then several python package like xlrd, xlwt, xlutils will be good enough. But I think a integrated environment will be more user friendly.

Unfortunately, there’s no alternative in OSX platform. Before OSX 10.6, we could use osxappscript to bridge Python to Excel, but Apple killed it to support its own AppleScript. Anyway, I shall start to work on the algorithm part instead of digging into too much of the debate of tools.

I am perfectly fine limited under windows platform. Among these candidates. Pyxll is adapted in business environment. I guess it is more reliable and well-organized. IronPython is an implementation of Python for Microsoft dot NET. thus I think too heavy.

I always follow the google python style guide. My Pyxll script is like this.

One thing to remind is that, to make your script available in PyXLL, you need to append it to the modules list in pyxll.cfg.
