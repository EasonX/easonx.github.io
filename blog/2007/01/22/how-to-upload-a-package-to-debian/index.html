How to upload a package to Debian

Create your gpg key

$ gpg --gen-key

Select the default and go on like that.

$ gpg --export --armor your-email-address > public-key-file

This file is needed while your submit a create acount application in
mentors.debian.net

Packaging;

Create an acount and upload it into mentors.debian.net

$ apt install dupload reportbug dput
$ vi ~/.dupload.conf

Add entry like:

and the in the directory where changes and dsc files in.

$ dupload -t mentors cream_0.32-2_i386.changesor

you can use an alternatives tool: dput

$ vi ~/.dupt.cf

and then type:

$ dput mentors cream_0.32-2_i386.changes
