人生若只如初见

题记：受好友所嘱，作文一篇，自高考之后，更无为他人所看之文，遍搜枯肠，行此题。古旧用变，聊付一笑。

夕阳没有什么余光，喧嚣的城市都只是眼前的一缕浮光。躲在身后的角落，笨笨迷茫的眼神中，冒出了一滴泪，悬在眶边上。W城，这个寂热的城市，就要离开了。

恰是一天前，笨笨怀着一腔的烦闷来到这个城市，本来这里是可来，可不来的，尤其是现在这个紧张的时刻，正处在人生的十字路口的他，应该是在与命运搏击，赢得人生的目标的时刻。但是，莫名的情愫，让笨笨告诉自己：这个时候，听听他的意见，一定是种正确的选择，毕竟，他不止是最好的朋友，还是知己，一些道不明的东西，驱使着他。

见到面的时候，没有想象的亲热，似乎越是熟识的人，越是拘谨。只是一个很随意的拥抱，然后笑笑：“还没吃吧？走，先去吃饭，我也饿了。”

一路上开着些不紧不松的玩笑，很快，就到了吃饭的地方。很不错，不像是学生应该去的地方。

“你小子挺小资的阿，是不是做干部被腐化了？天天来这种地方？”

“嘿嘿，我也是第二次来，上次我爸来的时候偶然发现的”……心里有些触动，不过还是一张笑嘻嘻的死鱼脸。对所受的一切处之泰然。笨笨知道，过分的拘谨对大家都不好。人是种很奇怪的动物，有时候知道怎样做才是合乎关系的，但是又期望着象普通人的一种认知。似乎熟悉和陌生只是一种很相对的概念，视乎心情，视乎环境。

“如果说我要是你，那么对最合适的出路只有一个：去香港。”

毕竟是soul mate级的人物，这么开门见山，笨笨什么都还没说，他就已经知道了来意。“我和你不同，你有隐忧，如果是美国，就算你的家人，我们，都放也心，你自己也不会放心。一旦出了什么事，怎么办？”

“所以说看看你的要求，和你自己最主要的矛盾，那么，都是说香港才是你最合适的选择，首先那里打的气候，饮食，和家里都差不多，其次，从未来的发展，我们都知道未来如果最合适的地方，应该还是国内，我们要的是见世面，是增长见闻。在这点上，香港很合适。”

“我以前也考虑过去德国，也和你提过，但是最后还是觉得国内对香港的认同比对德国强……”

“……”

这些考虑，肯定是来之前就想好了，对这个，笨笨更在意。

只有你care somebody的时候，才会琢磨这个人的目的，想法，才会去做些事情。我们，通常叫这种行为叫做关心。

本来笨笨还有些头疼，不知道该怎么开始这个谈话。笨笨不是一个很会说话，很能说话的人。但是以前人们总认为笨笨是个话多的人，家人还会用期待的语气说：“咱们家是要出个律师哦！”

话多，是为了掩饰孤单。

话多的人，往往都喜欢自顾自的讲话，他们怕看别人的眼神，怕自己的掩饰被识破。

就像是最自大的人往往内心深处极度的自卑一样，世界上的事，不外如是。

可是现在，这份担心就这么悄悄的烟消云散了，不露着一丝痕迹。我们对这种“解围”之类的行为都会生出一种特殊的感激，笨笨心想：“其实，感激的是理解吧~”

就如对面坐着的这个人，虽然是童年的玩伴，但是后来的玩伴一个又一个的换了，又怎么会如此特殊呢？记忆一点一点的滴入心头……

场景回到了那个纯真的年代。笨笨和他都还是孩子。

“你是自己把自己孤立起来，下围棋对你的意义就是这样吧？”

“一个人呆在没人的屋子里，琢磨没有人关心的东西，看别人看不懂或者不愿看的书，”

“其实，你是寂寞的吧~”

就像是用牛皮纸做胶布粘好的伤口被撕开了一样，有种被揭光了的恼怒。

笨笨很想发火，原本的生活尽管不甚如意，但总是自得其乐吧，连自己都从未也不愿思考的东西，对面这个人却直白的说了出来，就像是用别人的眼睛做镜子，看着光噗噗的自己，然后还满身伤痕。

可是,笨笨发出的声音却是低低的啜泣，这样的失态让自己难堪的很，在未来的生活中再也没有过了。

第二天，他们还是朋友，很好的朋友，只是，在笨笨的心里，他已经永远的不一样了。

世界上有三种题目，是所有的文学作品都扰不过的：爱情，母爱，生与死。在笨笨心中，却有些小小的腹诽：爱情固然激动人心，但是酒凉化茶，童话里的公主王子我们生活中也有：查尔斯,黛安娜，他们遇见，相爱，然后离婚；母爱无私而又坚忍，但是这种有着无奈代沟的温润之玉有时给的并不是我们想要或者需要的，爱到深处却又剑走偏锋，就像是练高深的内功，走火之后往往受伤越烈；只有生与死，是人类无法逾越的障碍。但是生，老，病，死，爱离别，怨憎会，求不得，乃是佛家七苦，苦痛之事展示于人，这是施虐抑或受虐？

只有一种感情，跨越时间，只会越陈越香，这种感情叫做：理解。

人最怕什么？

一千个人有一千个答案。

但人们都害怕一种东西：寂寞。多少人因为寂寞而谈恋爱，然后在伤了别人之后又伤了自己，又有多少人在寂寞中迷失自己，犯下难以弥补的过错？

我们年少轻狂，但我们不是坏人，我们也只是在追逐中希望填补那些寂寞的空缺而已。

寂寞，解药是理解。

笨笨的思绪被拉回到阳光下的W城，窗外行走的车子玻璃的炽烈反光告诉我们这是个多么美好的下午。这次来的目的已经得到了，但是笨笨心里总还是有些失落，不知道失落的是什么，只是有些莫名的空虚在侵蚀着内心中的某个角落。

笨笨的眼神只闪现了一星黯淡，之后却没事人似的，和其他同学依依惜别，中间还打趣了可怜的被到处枭首示众的Einstein，竟然在宿舍区也不能幸免。

只是， 她知道，他们已经永远不一样了，她已经不是那个坐在他身边的人了。这个位置有了新的主人。

一夜无言。只是，笨笨并没有睡着。明天还有明天的事，another day is just the same as usual。

第二天一早，好容易等到8点，去看看他，只见他睡在她旁边，早晨的阳光在床头，映照着他脸上，宛如圣婴。

“唉，还是这么懒床~”

笨笨无奈的笑笑，轻轻的关上了房门。嗯，让他睡吧，反正已经想好了，这次来的目的也达到了，虽然和所有给自己意见的人想的都不一样。但自己知道，这才是最理想的结果。至于其他的什么事，都是可办可不办的。

12点。看了一上午paper的笨笨知道，也到了该叫醒他的时候了。

惺忪的眼神，满脸的不好意思，嚅嚅的说到：

“昨晚上赶大试验的仿真，到四点半才睡，结果还是没能做完，让你等久了~”

“也是最近事太多了，申请之类的东西搞的太烦琐，又加上大试验~”

笨笨只是笑笑。

“没啥，下午怎么安排吧？”

在昨天的刚来的地方吃午饭，只是，这次是她请客。她落落如故，谈笑风生，一点没有刚认识应有的不自然，仿佛本来大家就是老朋友似的。

饭后：

“你对她是怎么想的？你出去，她怎么办？”

“你觉得呢？”

“其他我看不出，对一个人的了解就这么点接触，说出来太不负责。但是我知道，她真的很在乎你。”

“说真的，我也还是像假期和你去旅游时的想法一样，我觉得未来实在是不确定，我包不准自己会不会变，她也是。”

“她有些性格很配你阿，遇到这样的人真的很难得，而且她是真的对你动了情。只有对对方真正在意，才会在意他身边的人。”

“我现在还是有些矛盾吧，她还是愿意等我，但我自己的前途也还没有定。”

“相信我，好好待她，很多人，一旦失去就不再了。”

“……”该说的都说了，也是时候平静的回去了。

“干嘛这么急阿，反正回去也没有什么事？再玩两天吧，我大试验做完，就可以陪你逛逛W城了，W城人的聪明才智，都集中到发明小吃上去了~没试过可真可惜了~大试验今晚上肯定能搞完了~”

“呵呵，在外面久了，心里不踏实阿，有种负疚感呢~”

“我还是走吧，订上晚上的票，明天早上就能到了，周一还有组会呢，不要错过了。”

笨笨下决定很慢，但一旦下了决心，便是十头牛也拉不住的。她想回去，自然就会回去。

踏上归程，窗外是如时光一去不复返的街道。人人都在这夜色迷蒙中赶着自己的生活。眼前少了的这个人，也这样留在了热寂的城市的回忆里。

笨笨抱着胸前的装着衣物的鼓鼓的书包，底下了头，埋在包上，躲在角落里，静静的啜泣，生怕旁的人发觉。只是，旁的人有他们的生活，怎么会瞧陌生人的热闹呢？这些都是笨笨的多虑了。

从低低的啜泣声中，仿佛听到一句断断续续的模糊不清的词串：“我……我……也是……喜欢……你……你的……阿……”

上善若水

对任何一本西方哲学专著，都会提到泰勒斯，作为哲学的起源。我们不知道这个米利都人真正有些什么事迹，他就象我们传统的三皇五帝一样，很多事迹近乎神迹：例如，在那个年代（前6世纪），他预言了一次日蚀;在埃及，他应用了相似三角形的方法计算了金字塔的高度────也许在现在我们只会觉得这是至多一个初中生就能够作到的事────但，我们的古人在几个世纪之后，甚至是今天，仍然缺乏象他那样科学精神与思维方式。

作为希腊七贤之一，每个人都有一句出名的话语，传说，泰勒斯的话是：

“水为至善。”

怎么样？是不是很熟悉？在我们的春秋时代，也有一位国家图书馆的馆长说过类似的话：

“上善若水。”

不过东西方人思维的切入点虽然相通，但是思维的走向可就大相径庭;长耳馆长看到的是水的抽象性，也许，他只是恰巧的把“水”这个概念作为他令人难解的哲学的一个典型代表。但泰勒斯不一样。他不仅是个哲学家，并且也是一个科学家，虽然
在《原本》中的很多归功于他的东西也许是归错了，但他是那个年代最出色的科学家之一，这是毋庸置疑的。科学的实证主义和还原论的方法，在那个时代也许还没有提出，但是这种精神本就是科学的一部分，就象呱呱坠地的孩子，与生俱来的带着人类的好奇，人类的灵魂，精神。

泰勒斯把世界归结为水，不是说世界的本原，而是世界万物本身就是水。就连我们这个大陆，也是浮在无垠的水面上。

不由得想起来：

蒹葭苍苍，白露为霜。
所谓伊人，在水一方，
溯洄从之，道阻且长。
溯游从之，宛在水中央。
蒹葭萋萋，白露未晞。
所谓伊人，在水之湄。
溯洄从之，道阻且跻。
溯游从之，宛在水中坻。
蒹葭采采，白露未已。
所谓伊人，在水之涘。
溯洄从之，道阻且右。
溯游从之，宛在水中沚。

这突然的插叙，不完全是意识流的浮动，对那个年代，对这种人类文明初期的纯洁，对未来的冲劲，热情，对知识的渴望，以及这种贤人存在的可能────现代人，知识爆炸，已经不可能出现这样对一切主流学科都如此掌握的大家了，每个人能够在自己的行当里有所建树，已是了不起的人生────充满了憧憬。是泰勒斯是那位伊人？还是那个年代是那位伊人？这，并不重要吧。

感情往往产生自己的对立物，吝啬有时侯产生挥霍，挥霍有时侯产生吝啬，我们
往往由于脆弱而坚强起来，由于怯懦而大胆起来”

拉-罗煦福高《箴言集》182节

人们通常总是被自己亲身所发现的道理说服，更甚于被别人精神里所想到的道理

我常常想，自己是个多么怯懦的人，有着数不清的缺点，总是憎恶自己，又总是为
自己开脱──这不就是怯懦的最好表现么？──然而──又是可恨的然而──自己却又总认
为自己识见高人一筹，在明明的白日下，假装谦虚的说着“大家都一样，只是我想
的角度不一样罢了”;回到阴暗的角落，却又沾沾自喜于自己的与众不同。

间，事务等等的拖累，而无暇自省，并且还在于这个身体所能体会到的感受，经

因为明白，所以慈悲。只是，这种慈悲，或许又是自己脆弱的悄悄表现──我又原谅

父母都说我是没有童年的孩子，也许正是如此，所以仍然期盼童年，希望着自己就
是个孩子。我终究，并不是逃出了造物的圈套，只是陷得更深────或许，本来没有
什么圈套，这就是人生。

如何在linux下更新dell电脑的BIOS

$ sudo aptitude install libsmbios-bin libsmbios1
$ sudo getSystemId

得到结果：

Libsmbios: 0.13.6
System ID: 0x01B5
Service Tag: xxxxxxx
Express Service Code: xxxxxxxx
Product Name: MXC051
BIOS Version: A02
Vendor: Dell Inc.
Is Dell: 1

记住其中的system id：01b5。

3、浏览 网页
找到01b5的几个bios的目录。

system_bios_ven_0x1028_dev_0x01b5_version_a00/
system_bios_ven_0x1028_dev_0x01b5_version_a02/
system_bios_ven_0x1028_dev_0x01b5_version_a03/
system_bios_ven_0x1028_dev_0x01b5_version_a04/

我的是a02版本，所以先下a03目录中的bios.hdr

$ sudo modprobe dell_rbu
$ sudo dellBiosUpdate -u -f bios.hdr

重启后会自动更新。如果bios版本不对，系统会提示没有合适的bios.hdr文件，不会强制更新上去。更新必须一级一级的更新。例如我就更新了两次，从a02到a04。

更新有危险，尝试需谨慎。
