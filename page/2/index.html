Must know to write a grant proposal

I am currently writing a grant proposal. Yet it’s my first proposal. What’s the difference? I just randomly see this link. And write down here some points worth noting for myself.

“the abstract must sell the grant”

structure of an abstract:

  • brief background and problem
  • objectives and hypotheses
  • preliminary results
  • methodology and/or strategy
  • significance and innovation

types of abstracts

  • technical abstracts: for federal agencies;shorter backgrounds;elaborate on preliminary results, research strategies and specific aims.
  • lay abstracts: for private sectors; widely understandable; long background and significance sections.

avoid in abstracts

  • anything not covered in proposal
  • graphs/images, citations
  • wordy:
    • simpler vacabulary;
    • omit “overview” phrases;
    • shorten which/that clauses;
    • avoid there is/are or it is;
    • put the Action in to the Verb
    • untangling noun clusters
    • avoid interuption

checkpoints

  • does not omit any part
  • written with the reader in mind;
  • it meet the word count limitations;
  • its ideas are clearly expressed

Redirect my site after domain change

My domain will soon expired, and strangely, I cannot login to my GoDaddy account associated with Google App. My email to the customer service also got no response. Anyway, finally I decide to register another domain name, as it’s cheaper: 2.99 USD vs. 9.99 USD original one.

I just changed CNAME in my Github page, and I found the visit number drop rapidly after that transition. I NEED TO FIND A WAY TO REDIRECT OLD ENTRIES TO MY NEW DOMAIN.

Methods

First, Create a new repository, let’s say, panchoat.org. For the purpose of convenience, I create an automatically-generated Project Github page. Then on the web editing interface of this new repository, I added a file CNAME in the branch of gh-pages, which the Gitbuh pages is in. The content is my old site, panchoat.org, to tell the server this branch’s domain name.

Because Github Page does not support 301 redirect, I use this workaround to solve this problem: add both lines to index.html.

Then add a customized 404 pages, add these lines as well. This will ensure those from google search will directly lead to the new address.

Defense of PISA result of Shanghai

There are some criticisms for the top ranking of Shanghai in PISA 2012. For instance, time.com and slate.com.

However, some of the facts listed in those articles are misleading or even wrong at all.

  1. China is cheating because it only released the result of city of best performance.
    Ans: PISA is organized by OCED, all OCED countries participated in 2012 PISA. However, China is not an OCED member, and shanghai only participated as “an economy”. Nothing related to “manipulating the result”.

  2. Shanghai students performed better than other places, that’s why the college enrolling rate is higher than nationwide average.
    Ans: Firstly, enrolling rate is not 84%/24% (Shanghai vs nationwide). Nationwide rate is as high as 87% in 2012.. Secondly, indeed, students in Shanghai are commonly viewed as more easily going to elite Universities in China. But regional discrimination shall be the major cause, not the real performance. Shanghai parents are more affluent, but not necessary the wealthier their parents the better their performance academically.

Writing my own plotting package for R

I had introduced a previous blog entries talking about using R as figure generation engine. There are several advantages of using R:

  • scripts: this is the source of all the power of R. When using scripts, you can get rid of repeating operations.
  • high quality: nowadays , publication standard is 300 dpi. While it’s viable in other program, you still need to pay attention. DON’T USE SCREENSHOT.because windows limits the screenshot to 96 dpi. In R, a chunk option “dpi=300” is all you need to write.
  • learning curve: the learning curve is quite sharp, I have to admit it. But once you are comfortable with the design patterns, you will be more than happy to draw your figures.

My R figure settings and functions for publication basically are just using the general plot functions to draw time series and bar plots. But it requires some customizations to facilitate the processings.

references

Here I referred to the RStudio introduction page and viewed all 4 documents for package development. Among the three object-oriented models of R (S3, S4 and Reference Classes), I choose S4 as my skeleton.

First issue I encounterred is how to seperate my functions into ‘public’ and ‘private’ domains. The answer is in namespace. In my case, my namespace file looks like:

exportClasses("eucbars","euctraces","euclines")
exportMethods("draw")
export("eucbars","euctraces","euclines")

Then only these 3 classes and methods will be exported. I’m still polishing my code. Once after my paper gets accepted, I will release this package in this site.

How to revert to Windows 7 from Windows 8 on DELL Inspiron

It’s very annoying being unable to escape from a trap. One of my friend encountered a “trap”: Windows 8 on his DELL Inspiron laptop. After a few googlings a trials, I think I could lend him a hand to escape from that trap.

Windows 8 boot options are changed. So that we cannot press F8 on the power-up screen and the legacy “DELL Restore to Factory” option is no more existed now.

The trick is a command line tool named imagex. Microsoft has a detailed description of this command.aspx). The hotfix update will contain this tool.

Use either windows PE boot disk, containing this tool; or the “Recovery of your windows” option after you install this tool. Locate your Recovery partition. It should be noted that the image file factory.wim is a file residing in ‘Dell\Image’ path. Both directories are hidden.

If your OS partition is F: (in my case), the recovery partition is C:(also in my case) the command will be:

$ imagex /apply C:\Dell\Image\Factory.wim 1 F:

Then after around 15 mins. the recovery will be finished and you can reboot and enjoy the factory windows 7 now.
